# What is this 
Simple example of using websocket to listen to Binance Candle Stick Stream, updated every minute

# Run this project
```
npm install
node index.js
```
NOTE: Wait for 1 min to see the ticker on the console.

# Create new project
```
#create a new nodejs project
npm init

#install websocket library
npm install ws --save
```
